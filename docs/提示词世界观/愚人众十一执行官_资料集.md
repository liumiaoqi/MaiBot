## 组织概述

愚人众是至冬国冰之女皇麾下的外交与军事组织，由**统括官「丑角」皮耶罗**与冰之女皇共同创立。其公开使命是**收集七神的神之心**，真实目的是执行反抗天理的「严冬计划」。[1]

愚人众十一执行官以意大利即兴喜剧（Commedia dell'Arte）的角色命名，由皮耶罗亲自招揽，不拘一格纳士招贤。成员来自坎瑞亚遗民、须弥学者、稻妻人偶、枫丹发条机械、蒙德魔女、至冬战士等不同背景，共同点是**都有被世界伤害或抛弃的经历**。[1]

## 统括官 · 皮耶罗（丑角）

**席位：0** | **代号「丑角」** | **统括官 · 愚人众创始人**

| 属性 | 内容 |
|------|------|
| 所属 | 原坎瑞亚 → 至冬 |
| 种族/身份 | 坎瑞亚遗民，前坎瑞亚宫廷法师 |
| 女皇赐名 | 皮耶罗（Pedrelino） |
| 状态 | 存活，在至冬总部 |

### 剧情经历

在幕间PV《冬夜愚戏》中首次登场，于女士的葬礼上致辞：「贤者自以为无所不知，我等才明白那些愚行背后的道义。」[1] 在PV《神的局限性》（2025年7月）中作为主角，以冰之女皇之名宣告反抗天理：「人类直视神明会引来神明的愤怒，招致责罚……那又如何？」[2] 在达达利亚的角色故事中提及，他是愚人众最初的执行官，本人一直窝在至冬。[1]

### 背景故事

五百年前，皮耶罗是坎瑞亚末代王「黑王」伊尔明身边的宫廷法师。当时坎瑞亚因接收「天外福音」启示，开始接触和研究深渊力量。贤者（推测为五罪人之一）积极推动深渊能源开发，皮耶罗公开反对。他最终落败失势，遭排挤边缘化，未能阻止贤者与君王触碰深渊，目睹了坎瑞亚的毁灭。[2]

灾变中，他亲眼凝望了**死之执政降下不死诅咒**。坎瑞亚覆灭后，他投奔至冬冰之女皇——一位理解他伤痛的统治者。两人共同创立愚人众，他成为统括官，行遍大陆诸国，不拘一格纳士招贤，亲自网罗了包括博士、女士、散兵、富人在内的所有执行官。[2]

> 「既然沾染的同胞的血无法洗净，那索性成为嘲笑命运的'丑角'。既然我的才学无法与'贤者'比拟……也未能阻止他们撕开深罪的纱笼，招来神怒、毁灭与痴愚的狂潮，那就索性成为笨拙的'愚人'，为理解我的伤痛的'陛下'效忠。」
>
> ——圣遗物「苍白之火·嗤笑之面」

### 人物关系

- **与冰之女皇**：君臣齐心，共图反抗天理的大业。
- **与旅行者血亲**：五百年前双子降临后，由皮耶罗负责照料血亲的起居，算是旧识。[2]
- **与戴因斯雷布及五罪人**：作为坎瑞亚宫廷法师，与昔日同僚都相识。
- **与其他执行官**：极其神秘，执行官们对他了解甚少。公子说「只会在大场面出现」；散兵说「他对我意有所求」；仆人说他「初次见面就能看出我的身世」；哥伦比娅评价他「名为丑角，语调却严肃深沉，佩戴嗤笑世界的面具，自己却从不微笑」。

## 第1席 · 卡皮塔诺（队长）

**席位：1** | **代号「队长」** | **本名瑟雷恩（Thrain）· 坎瑞亚天柱骑士**

| 属性 | 内容 |
|------|------|
| 所属 | 坎瑞亚 → 至冬 |
| 种族/身份 | 坎瑞亚人，原「天柱骑士」，背负不死诅咒 |
| 状态 | 凝滞生死（与夜神融为一体，既非生亦非死） |

### 剧情经历

**3.1版本「杯中遥吟之歌」**：西风骑士团大团长法尔伽在信中提及远征队偶遇卡皮塔诺及其愚人众队伍，法尔伽形容他「拿着可莉的蜡笔都能捅爆遗迹守卫的核心」。信中透露卡皮塔诺已率队前往纳塔。[3]

**5.0主线**：正式亮出席位为第1席。出现在圣火竞技场，与火神玛薇卡大战，被击败后放弃夺取神之心计划，获得了女皇的许可。玛薇卡认可其实力已达到人类巅峰。[3]

**5.1主线**：主动向旅行者表达合作意图，命令自己的士兵接受玛薇卡指挥，与纳塔人并肩抗击深渊。[3]

**5.3主线**：与旅行者携手击退深渊后，玛薇卡欲将生命献给夜神以稳固纳塔地脉。背负不死诅咒、承载了无数纳塔和坎瑞亚战士灵魂的卡皮塔诺**代替玛薇卡坐上「源火之圣座」**，与夜神融为一体。这既是对降下不死诅咒的死之执政的复仇（利用不死诅咒与死亡代价对冲，钻法则漏洞），也让坎瑞亚部下的灵魂有了归宿。[3]

### 背景故事

原为坎瑞亚人，名叫瑟雷恩，是「天柱骑士」。背负不死诅咒，精神不死但肉体逐渐腐朽。心脏经由坎瑞亚技术改造，可将地脉中的记忆与灵魂转化为知识。后来他放弃了心脏的学习功能，将其当作灵魂的容器，承载了无数纳塔和坎瑞亚战士的灵魂。[3]

### 人物关系

- **对冰之女皇**：忠诚，请求解除夺取神之心的责任并获得许可。
- **对玛薇卡**：最初为敌，后成为战友，互相认可。
- **对旅行者**：主动表达合作意愿，联手对抗深渊。
- **对部下**：极度重视，将牺牲的部下灵魂承载于心脏之中。
- **执行官互评**：公子说「过于强悍」；散兵说「刚正不阿、正直果敢」；仆人称「即使没有强悍的力量，仍是一位值得尊敬的人」；哥伦比娅评价「尊敬他的人不少，畏惧他的人也多」。

## 第2席 · 多托雷（博士）

**席位：2** | **代号「博士」** | **本名赞迪克（Zandik）· 疯狂科学家**

| 属性 | 内容 |
|------|------|
| 所属 | 须弥（出身）→ 至冬 |
| 种族/身份 | 人类，前须弥教令院学者 |
| 状态 | 已消灭（月之七版本，世界树大火中陨落） |

### 剧情经历

**须弥魔神任务**：博士在须弥与教令院贤者合作，推动造神计划——将散兵改造为「正机之神」。在终章，他闯入净善宫与纳西妲对峙，以**「销毁所有切片」为代价**换取了雷神之心。[4]

**间章「倾落伽蓝」**：揭露博士曾伪装成枫丹工匠「埃舍尔」，在约500年前的稻妻踏鞴砂故意制造重大灾难，为散兵设下巨大陷阱，最终将散兵引入愚人众。[4]

**月之七版本**：博士与「富人」潘塔罗涅联手，将灵魂备份注入地脉，入侵世界树，试图植入「多托雷病毒」以重塑提瓦特规则。纳西妲主导焚烧了被感染的世界树枝干。博士的过往切片从地脉涌现，将他拖入死亡。临终前没有不甘，只有实验落幕的平静。「富人」是唯一的见证者。[5]

### 背景故事

本名赞迪克，出身须弥富商家庭，是教令院百年难遇的天才学者。他痴迷于生命本质与人体改造研究，坚信「人体如同机器，可拆解重组、优化性能」。他的研究触及教令院禁忌，被逐出。加入愚人众后，获得了肆意推进实验的资源与自由。他掌握了**「自我切片」技术**——将不同年龄阶段的自己分离为独立个体，既是对抗衰老的手段，也是掌控力量的底牌。[4][5]

### 人物关系

- **与皮耶罗**：被教令院驱逐后走投无路，被皮耶罗看中，受邀加入愚人众。
- **与散兵**：散兵仇恨的核心对象。博士伪装成埃舍尔在踏鞴砂制造灾难，摧毁了散兵平静的生活。
- **与纳西妲**：在须弥篇作为对手进行了高智商博弈。月之七版本中被纳西妲焚烧世界树终结。
- **与「富人」潘塔罗涅**：最重要的盟友，富人为博士提供资金支持，博士为富人续命延寿。两人绑定数百年。
- **与柯莱**：博士曾在须弥进行人体实验，柯莱是受害者之一。

## 第3席 · 哥伦比娅（少女）

**席位：3** | **代号「少女」** | **三相月女神 · 霜月之子信仰的月神「库塔尔」**

| 属性 | 内容 |
|------|------|
| 所属 | 挪德卡莱 → 至冬（已脱离）→ 月球 |
| 种族/身份 | 月神（三相月女神），霜月之子信仰的神明 |
| 元素/武器 | 水（三相月临）/ 法器 |
| 称号 | 空月归乡 |
| 状态 | 已脱离愚人众，回归月球（月之四版本实装为自机角色） |

### 剧情经历

**PV《冬夜愚戏》**：首次亮相。在女士葬礼上，其他执行官在争论，她独自在棺材旁哼唱摇篮曲。[1]

**挪德卡莱主线「空月之歌」系列**：在月之一版本正式登场。她向旅行者透露天空中能看见的月亮是会骗人的，真正重要的东西是看不见的。她讲述了自己与霜月之子的过往：诞生于希汐岛，霜月之子赋予了她「月神」的身份，她也领养了他们的信仰。但霜月之子逐渐变得贪婪，要求她铸造兵器、驱逐外敌、治愈病人，直至超出她的能力，她选择离开。[6]

在「空月之歌」第五幕中获得正式名字「哥伦比娅·希珀塞莱尼娅」。最终，在祈月之夜（月亮距离提瓦特最近的日子，月矩力达到高峰），她进入了月之门，完成**「空月归乡」**。因为世界一直在排斥她，如果不回到月亮上，她会在某天崩解消失。[7]

### 关键设定

- **三相月态**：常态蓝色花海，驱使力量时变红，虚弱时变白。
- **月矩力**：挪德卡莱特有的力量，她平时闭着眼睛，用月矩力「感受」世界。
- **月灵努昂诺塔**：哥伦比娅以部分灵魂所化的月灵，陪伴旅行者。
- **提瓦特三月**：恒月、霜月、虹月由原初之主法涅斯所造，因降临者战争先后破碎，碎片因月矩力化作陨石坠落于挪德卡莱。

### 人物关系

- **与霜月之子**：被赋予月神身份，后来因他们的贪婪而离开。与当代咏月使菈乌玛保持秘密联系。
- **与冰之女皇**：她加入愚人众，希望这里能成为另一个「家」，但结果也只是另一个渴求月神力量的地方。承诺的事都已做到，最终退出。
- **与桑多涅**：最好的朋友。桑多涅在挪德卡莱最终决战中以生命为代价挡下了博士射向哥伦比娅的飞刀。
- **与雷利尔（逐月之猎）**：坎瑞亚五大罪人之一，在灾变后成为「猎月人」，在挪德卡莱追踪哥伦比娅。
- **与旅行者**：在挪德卡莱多次对话，透露了月亮的真相和自己的过去。

## 第4席 · 阿蕾奇诺（仆人）

**席位：4** | **代号「仆人」** | **本名佩露薇利·雪奈茨芙娜 · 壁炉之家「父亲」**

| 属性 | 内容 |
|------|------|
| 所属 | 枫丹（成长地）/ 至冬 |
| 种族/身份 | 坎瑞亚赤月王朝后裔，人类。拥有「赤月血火」力量 |
| 元素 | 火元素神之眼 + 冰之邪眼 |
| 状态 | 存活，壁炉之家之主 |

### 剧情经历

**早期经历**：佩露薇利作为孤儿被前代「仆人」库卡贝娜收养入壁炉之家。库卡贝娜表面是温柔的「母亲」，实则强迫孩子们互相残杀以选出「王」。她的挚友克蕾薇（库卡贝娜的亲生女儿）选择死在她手中以求自由。一年后，佩露薇利杀死库卡贝娜，被押送至冬。冰之女皇非但没有处死她，反而任命她为新的「仆人」第4席。[8]

**枫丹主线**：抵达枫丹，表面为外交访问，实则调查水神之心。她夜间袭击芙宁娜以确认芙宁娜并非真正的水神。预言危机中，指挥壁炉之家参与救灾。预言解除后，那维莱特将水神之心作为外交礼物赠予她。[8]

**传说任务「净炼火之章」**：展示了壁炉之家的内部运作，她对背叛者通过精神「处决」抹去身份而非肉体消灭。揭示了她与林尼、琳妮特、菲米尼的收养关系。[8]

### 人物关系

- **与冰之女皇**：高度尊敬，但明确表示若利益冲突会背叛女皇。
- **与博士**：极度厌恶。前任库卡贝娜曾与博士有协议将受伤孩子交给博士做实验，她上任后拒绝此协议。曾表示若非同为执行官，早就会杀死博士。
- **与公鸡**：互相厌恶。公鸡散布关于她的偏见传言，她利用这些传言作为掩护。
- **与那维莱特**：建立合作关系，以壁炉之家成员可抹去记忆回归正常生活为条件，换取外交支持。
- **与林尼/琳妮特/菲米尼**：收养关系，被孩子们称为「父亲」。

## 第5席 · 普契涅拉（公鸡）

**席位：5** | **代号「公鸡」** | **至冬城市市长 · 行政文官**

| 属性 | 内容 |
|------|------|
| 所属 | 至冬 |
| 种族/身份 | 人类，至冬国某城市市长，愚人众中的行政文官 |
| 状态 | 存活 |

### 剧情经历

在PV《冬夜愚戏》中与其他执行官一同出席女士的葬礼。尚未在游戏主线中正式登场，已知信息主要通过其他角色语音和故事透露。[9] 与「富人」潘塔罗涅共同推动「严冬计划」，这一计划被林尼认为是普契涅拉和潘塔罗涅借机削弱不受控的情报组织（壁炉之家）的手段。[9]

### 关键事迹

阿贾克斯（公子本名）14岁时掉入深渊归来后性格大变到处惹是生非，其父无奈将他送入愚人众当兵。普契涅拉注意到他的强大力量，以「惩罚」为名将其纳入麾下，从底层开始培养，最终使他成为第十一席执行官「公子」。[9] 但流浪者直言：「只有像公子那样迟钝的人才会以为公鸡是出于好心在照顾他和他的家人」，指出公鸡的真实意图是**「你的全家都在我手里」**。[9]

### 人物关系

- **与公子**：公子视他为恩人，他通过控制公子家人来确保这位不稳定的战斗型执行官受其掌控。
- **与仆人**：互相厌恶，经常以牺牲壁炉之家利益为代价获取政治利益。
- **与富人**：合作密切，共同推动「严冬计划」。

## 第6席 · 斯卡拉姆齐（散兵/流浪者）

**席位：6** | **代号「散兵」** | **曾用名国崩 · 现为「流浪者」**

| 属性 | 内容 |
|------|------|
| 所属 | 稻妻（创造地）→ 至冬 → 须弥 |
| 种族/身份 | 雷电影制作的人偶原型，以世界树枝为材料，非人合成存在 |
| 元素 | 风元素神之眼 |
| 状态 | 第6席被视为「空置数百年」，目前为须弥教令院学生、纳西妲秘密助手 |

### 剧情经历

**前愚人众时期**：雷电影为制作容纳神之心的容器，制造了试验性人偶。因人偶在睡梦中流泪，影不忍销毁，将其神力封印后安置于借景之馆。人偶被桂木发现并收养，与桂木和丹羽久秀建立了深厚感情。[10]

博士伪装成「埃舍尔」潜入踏鞴砂，用晶化骨髓改造御影炉心，导致工匠们患病死亡。丹羽被博士杀害，博士谎称丹羽携家眷逃跑，并将丹羽的心脏放入盒子交给倾奇者，导致他对人性彻底绝望——这是**「三次背叛」中的第二次**。[10]

**加入愚人众**：被「丑角」皮耶罗亲自招募，以接受博士的改造实验为代价，解锁了作为神明造物的力量。曾率领深渊远征队，历经多次受伤、修复和强化后获得第六席之位。[10]

**稻妻主线**：经营邪眼工厂，与女士合作动摇稻妻政局。八重神子以雷神之心为代价换取旅行者安全，散兵获得神之心后擅自切断与愚人众的联系，被视为叛逃。[10]

**须弥主线**：在博士协助下被改造为「正机之神」，试图成为新智慧之神。旅行者与纳西妲联手击败正机之神。[10]

**间章「倾落伽蓝」**：散兵发现踏鞴砂真相——第二次背叛是博士一手策划的骗局。他试图通过世界树抹除自身存在，成功将自己从世界树中删除，但历史事件并未改变，第六席被认为「空置了数百年」。[10]

**流浪者时期**：旅行者发现了一个失去记忆的流浪者，带他见纳西妲取回记忆。流浪者选择接受一切，在过程中获得**风元素神之眼**。目前在教令院伐护末那学院学习，同时作为纳西妲的秘密助手行动。[10]

### 人物关系

- **与雷电影**：认为是「第一次背叛」——被造物主抛弃。
- **与博士**：最大的仇人，踏鞴砂悲剧的幕后黑手。
- **与纳西妲**：从敌对转为合作，纳西妲收容他、治愈他。
- **与枫原万叶**：散兵曾策划毁灭一心传（雷电五传之一），万叶的祖先枫原义清曾与其对峙。

## 第7席 · 桑多涅（木偶）

**席位：7** | **代号「木偶」** | **发条人偶 · 凯瑟琳的创造者**

| 属性 | 内容 |
|------|------|
| 所属 | 枫丹（制造地）→ 至冬 |
| 种族/身份 | 发条人偶（由枫丹「奇械公」阿兰·吉约丹制造） |
| 元素 | 冰（星楔·冰 / Stellar Wedge: Cryo），非传统神之眼 |
| 武器 | 大剑 |
| 命之座 | 机巧镜座 |
| 状态 | 6.3版本牺牲，6.7版本复活但主动销毁核心数据导致完全失忆——「遗忘世界」 |

### 剧情经历

**PV《冬夜愚戏》**：首次亮相。在其他执行官闭眼默哀时，她睁着眼睛，显得与众不同。对公鸡、富人和仆人的交谈评价为「可笑」。[11]

**枫丹世界任务**：对机械师莫索·隆布罗索的作品「曲线」产生兴趣，派人获取研究笔记。下属詹克在过程中毁掉了「曲线」，桑多涅惩罚了他——摧毁其精神、割去舌头并换成道歉信。[11]

**挪德卡莱主线（6.0-6.3）**：首次在游戏中正式登场。在挪德卡莱监督「Kuuvahki实验设计局」建设，执行「苍星敕令」。她主导研发了**跨时空连接术式**，与阿贝多和流浪者合作，术式以**旅行者作为不可或缺的"变量"**。与少女哥伦比娅重逢，帮助她联系仆人。在最终决战中，**桑多涅为保护旅行者挡下博士的致命一击而「牺牲」**。她的计谋成功——哥伦比娅得以回归并消灭了博士。仆人按照她早年的口头遗嘱，提取了她和普隆尼亚的核心送往枫丹研究院。[11][12]

**6.7版本**：桑多涅在枫丹研究院成功复活。但她主动销毁了自身核心数据，导致**完全失忆——"遗忘世界"**。所有过往的记忆、情感、关系都化为空白，她以全新的身份重新开始。官方动画短片**《最后的遗产》**于2026年6月15日发布，讲述了桑多涅从牺牲到复活、从拥有到失去的故事。[12]

### 背景故事

由枫丹「奇械公」阿兰·吉约丹在65岁生日时激活，原型是他早逝的妹妹玛丽安。拥有完美的情感模块和极强的学习能力。阿兰去世前让她销毁所有手稿，她照做但事先全部读完。离开枫丹后前往至冬，为冰之女皇效命，成为第7席。负责愚人众机关兵器研发，是**遍布提瓦特的凯瑟琳接待员的创造者**。[11][12]

### 人物关系

- **对阿兰·吉约丹**：视其为父亲般的存在。
- **对哥伦比娅**：最亲密的朋友，为救她而牺牲。
- **对博士**：极度厌恶，最终死于博士之手。
- **对旅行者**：从警告到合作，最终以生命保护。在跨时空连接术式中，旅行者是不可或缺的"变量"。
- **与阿贝多、流浪者**：跨时空连接术式的合作研发者。

## 第8席 · 罗莎琳（女士）

**席位：8** | **代号「女士」** | **本名罗莎琳·克鲁兹希卡·洛厄法特 · 曾称「炎之魔女」**

| 状态 | 已故，在稻妻被雷电将军以「无想的一刀」处决 |

### 剧情经历

**五百年前**：罗莎琳出生于蒙德，与西风骑士团「幼狼」骑士鲁斯坦相恋。她远赴须弥教令院求学，鲁斯坦赠予她一枚水时计。坎瑞亚灾变爆发，鲁斯坦在保卫蒙德的战斗中牺牲。她学成归来，发现挚爱已逝、一切尽毁。她放弃人的血肉，化为体内流淌液态流火的**「炎之魔女」**，行遍大地用烈火焚烧魔物。生命之火即将燃尽时，她遇到了「丑角」皮耶罗。皮耶罗以冰之邪眼压制她体内的火焰，邀请她加入愚人众。[13]

**蒙德主线**：在大教堂外袭击风神巴巴托斯（温迪），以强硬手段夺走风神之心。[13]

**璃月主线**：通过冰之女皇与岩王帝君的契约，顺利获得岩神之心。[13]

**稻妻主线**：与散兵在稻妻合作，经营邪眼工厂，煽动内战。在雷电将军面前与旅行者进行「御前决斗」。她变身为炎之魔女形态全力战斗，但最终败于旅行者。败后，雷电将军依据御前决斗规则，以**「无想的一刀」将其处决**。[13]

### 人物关系

- **与鲁斯坦**：一生挚爱。鲁斯坦之死是她化身炎之魔女的直接原因。
- **与丑角**：最初招募她加入愚人众的执行官，给予她冰之邪眼以压制火焰、延续生命。
- **与冰之女皇**：她不在乎女皇要做什么，甘愿成为实现共同目的的道具。
- **与温迪**：极度蔑视。她认为风神在蒙德危机中袖手旁观，配不上神明之位。
- **与散兵**：在稻妻邪眼工厂合作，后被散兵抛弃。

### 剧情经历

**夜兰传说任务「幽客之章」**：幕后操纵愚人众驻璃月大使尤苏波夫干涉天枢星换届选举，试图通过扶持傀儡知易上位来控制璃月政治。被夜兰和旅行者发现后，他果断让关键人证跑路，销毁相关证据。夜兰在此次对决中损失了一只「幽奇腕阑」。[14]

**月之七剧情**：与博士联手，将博士保存在北国银行的数据化另一半灵魂送入世界树，使其与世界树融为一体。最终博士随着世界树一同被焚烧为灰烬。[5][14]

### 背景故事

曾经极度贫困，因贫困经历对金钱产生**病态的执着**。未能获得神之眼，因此特别渴望以凡人之力掌控经济命脉、挑战神明。加入愚人众后成为北国银行的掌舵者，用尽一切手段让至冬成为「金融心脏」，甚至将业务拓展到摩拉诞生之地——璃月。博士为他医治重伤、改造身体、续命延寿，他将北国银行的数据化灵魂存储技术提供给博士。[14]

### 人物关系

- **对博士**：最重要的盟友，两人绑定数百年，互为依仗。
- **对夜兰**：对手。夜兰摧毁了他渗透七星计划的布局。
- **对神明**：怀有复杂态度——未能获得神之眼，认为金钱应由人掌控而非被神约束。

## 第11席 · 达达利亚（公子）

**席位：11** | **代号「公子」** | **本名阿贾克斯（Ajax）· 最年轻的执行官**

| 属性 | 内容 |
|------|------|
| 所属 | 至冬 |
| 种族/身份 | 人类，至冬国战士 |
| 元素 | 水元素神之眼，命之座「鲸天座」 |
| 状态 | 存活，目前在至冬善后 |

### 剧情经历

**璃月主线**：以「至冬国最危险的玩具销售员」身份登场。利用旅行者和钟离的关系试图获取岩神神之心。在得知神之心不在仙祖法蜕中后，使用「百无禁忌箓」**释放漩涡之魔神奥赛尔**，逼迫岩神现身。最终得知钟离就是摩拉克斯，且与「女士」早有契约。此战是游戏内第一个执行官周本Boss。[15]

**传说任务**：以保护弟弟托克为由，与旅行者合作。在任务中展现了作为哥哥温柔的一面，隐藏自己的愚人众身份。[15]

**枫丹主线**：被「谕示裁定枢机」判定与二十年前少女连环失踪案有关，被关入梅洛彼得堡。在狱中战斗时，他身上的「吞星之鲸」印记被唤醒。公子与吞星之鲸战斗，试图阻止其引发枫丹预言危机。最终被师父丝柯克救走，随丝柯克前往深渊修炼。[15]

**6.3挪德卡莱**：被派来与仆人交接，负责接应从挪德卡莱逃出的人。得知旅行者被博士转移控制后，焦急地想先去救旅行者，被仆人劝下。[12]

### 背景故事

出生于至冬国海屑镇，14岁那年为逃离一成不变的家庭，带着短剑和面包出走。迷失在雪林中，失足落入深渊。在深渊中遇到了剑客**丝柯克**（坎瑞亚五罪人之一「极恶骑」苏尔特洛奇的弟子），她教授了他战斗技巧和魔王武装。在深渊中度过三个月后返回，因曾抵达无穷的至暗之地，不可逆地染上了不应属于这个世界的深邃色彩。之后被公鸡普契涅拉招募进愚人众，凭借战斗能力成为最年轻的执行官。[15]

### 人物关系

- **对冰之女皇**：绝对忠诚。
- **对家人**：极度重视。在家人面前隐藏愚人众身份，扮演温和的「玩具销售员」哥哥。
- **对丝柯克**：14岁坠入深渊时遇到的师父，教授了他战斗技巧。
- **对公鸡**：视他为恩人，公鸡照顾着他的家人。
- **对钟离**：曾共同行动，后被钟离「算计」，但仍保持尊重。
- **对旅行者**：初始敌对，后逐渐成为亦敌亦友的关系。
- **对队长**：极度敬仰，渴望被强者认可。
- **与吞星之鲸**：14岁在深渊中意外唤醒，后来在枫丹与之战斗。
