# 第一篇：世界观总论

## 1. 虚数之树与量子之海

崩坏宇宙的底层逻辑是**虚数（存在）之树**与**量子（虚无）之海**的永恒角力，以哲学的角度看便是存在与虚无的对抗。

**虚数之树**
- 遍入一切的存在本质，是整个宇宙的本体结构
- 树的枝叶承载着无数世界，每一枚"叶子"都是一个宇宙或星系
- 各枝叶之间有巨大的虚数能量阻隔，使得大多数世界彼此孤绝、无法交流
- 虚数之树会吸收量子之海的信息（记忆），生长出新的枝叶（星系）
- 命途力量本质上就是虚数能量的一种表现形式

**量子之海（忆质之海）**
- 与虚数之树永恒对抗的存在，试图溶解虚数之树
- 虚数之树毁灭后的枝叶会掉入量子之海，变成信息碎片
- 量子之海中的世界称为"世界泡"，大多空间、时间混乱
- 除非有特殊装置稳定时空，否则世界泡中很难生存
- 在虚数之树中陷入量子纠缠状态就有概率坠入量子之海

**亚空间场与忆域**
- 除虚数之树和量子之海外，还存在着亚空间场和忆域等其他未知空间
- 忆域是记忆星神浮黎所掌管的空间，与记忆和过去有关

## 2. 宇宙的空间结构

崩铁宇宙有着严密的层级结构，由小到大依次为：

| 层级 | 说明 |
|------|------|
| 小行星/小行星群 | 宇宙中到处都是，小行星群是丰富的资源群 |
| 卫星 | 星球的卫星 |
| 次级行星 | 拥有D级以上文明 |
| 行星 | 分为固态和气态 |
| 巨行星 | 大型行星 |
| 恒星 | 包括红矮星、蓝巨星、红巨星等 |
| 恒星系 | 最小的恒星系统，内部拥有未知数量的恒星 |
| 星团 | 恒星系的直接上级结构 |
| 星云 | 大型星云结构，内部可能存在中子星 |
| 星系 | 真正的星系，拥有旋臂，河系大小 |

**重要的宇宙结构**
- **天彗星墙**：以光年为单位的巨大结构，包裹着文明居住的区域
- **亚空晶壁**：同样以光年为单位，是另一种宇宙屏障
- **巨引源**：超星系团才会存在的特殊结构

## 3. 命途系统

**命途**是星神所走的成神之路，每一位星神的诞生都意味着一道命途的开启。命途力量本质上就是虚数能量。

**命途行者**：有意或无意踏入或践行星神执掌命途的人。任何人都可以成为命途行者，只要其行为符合某条命途的理念。

**令使**：践行命途的行为得到星神认可，被赐予力量的代行者。令使拥有远超普通命途行者的力量。

**派系**：以星神为信仰聚集形成的组织。

## 4. 星核

**星核**被称为"万界之癌"，是毁灭星神纳努克散播的灾祸源头。

- 星核会引发各种危机，导致世界陷入混乱
- 星核的出现会进一步加剧星际旅行的难度，阻断星轨
- 游戏中开拓者需要解救由星核引发的种种危机
- 开拓者体内封印着星核，这也是其特殊能力的来源之一

## 5. 裂界

**裂界**是随星核而出现的空间扭曲现象，不停蚕食着现实世界。裂界中会出现各种异常生物和扭曲的空间结构，对现实世界构成巨大威胁。

# 第二篇：宇宙历史大事记

崩铁宇宙的历史由**琥珀历**记年。以下是宇宙中发生过的重大事件：

## 一、黄昏战争

**破坏力**：未知 | **影响力**：催生了多个星神

在星神大规模出现前，宇宙中游历着众多名叫**古兽**的远古生命。它们吞噬星体，使宇宙趋于混沌。之后诞生了几个星神（可能有存护星神克里珀和不朽星神龙），与古兽展开大战。最终古兽几乎灭绝，仅剩一个古兽升格为了**贪饕星神奥博洛斯**。

## 二、寰宇蝗灾

**破坏力**：三分之二有生区域 | **影响力**：秩序和繁育星神死亡

众多星神接连亮相后，**不朽星神龙**不知为何消失。对文明社会影响最大的是**秩序星神太一**——祂建立秩序使社会发展稳定，但有极强的统治掌控欲，想消灭一切风险。

一位征服世界的"王"被**虚无星神IX**无意义瞥视，成为自灭者。他截获了**终末星神末王**的预言，来到遍地虫子的蠹星，试图根据预言建立帝国。

**秩序星神太一**也得知了预言，试图阻止灾难，一边引导"王"杀光虫子，一边引导赏金猎人来到蠹星。但这些行为反而加速了**繁育星神塔伊兹育罗斯**的诞生，导致寰宇蝗灾爆发。

繁育星神化为自我繁殖的狂潮，污染侵略无数生命世界。**贪饕星神奥博洛斯**乘机吞入被毁灭的世界。

最终，**秩序星神太一**与**存护星神克里珀**做了交易，克里珀带领众多星神将繁育命途敲碎，打跑了奥博洛斯。而太一则被**同谐星神希佩**吞噬。

## 三、第一次繁荣、边星贸易战争、帝皇战争

**破坏力**：半数有生区域 | **影响力**：有机与无机生命的矛盾

大派系星际和平公司初步形成，**开拓星神阿基维利**几乎联通了所有世界，天才俱乐部使得科技爆炸，宇宙进入第一次繁荣。

公司的资本化运作打破了宇宙的均衡，**均衡星神互**拨动了宇宙的天平，**边星贸易战争**爆发，无数公司势力的边境世界开始反抗。

公司内部一个部门为了积累资本，开拓了无机世界，却意外唤醒了天才**帝皇鲁伯特**。鲁伯特作为无机生命认为有机生命会毁灭宇宙，于是发明**〈反有机方程〉**传播在各个无机世界，**帝皇战争**爆发。

帝皇战争后期，天才**寂静领主波尔卡**出手，利用自己的能力杀死了帝皇鲁伯特，帝皇战争结束。

## 四、第二次繁荣、星际能源战争

**破坏力**：几乎所有文明世界

宇宙进入休养生息期，公司建立资本主义帝国，宇宙进入第二次繁荣（又称"血锦之纪"）。后期**智识星神博识尊**突然停止了计算，原因不明。

之后由于资源分配和公司资本运作问题，**星际能源战争**爆发，各个文明各自为战，破坏力不亚于寰宇蝗灾和帝皇战争。

## 五、第二次帝皇战争、学派战争

星际能源战争末期，一位有机生命意外感染了**〈拟造反有机方程〉**，找到了帝皇鲁伯特的墓地，继承了其所有智慧和资产，成为**鲁伯特二世**，发动第二次帝皇战争。

鲁伯特二世沉迷于制造外置思维计算机**〈权杖〉系统**，意图利用权杖升格为完美学者，但被寂静领主波尔卡杀害。权杖系统被公司收缴交由博识学会处理。

博识学会内部因权杖算力分配问题爆发**学派战争**。一位叫做帕提维娅的普通智识行者在波尔卡的影响下找到了权杖中心，选择觐见博识尊，最终榨干了权杖系统算力，波尔卡借此破坏了权杖系统。

这一切都是**智识星神博识尊**计算干涉引导的结果——祂早在很久之前就计算出了结果，所有权杖的算力本质上都是为博识尊提供。

## 六、巡猎丰饶战争

**破坏力**：仙舟和丰饶星球 | **影响力**：威慑了求药使

星际能源战争结束后，仙舟文明最初是作为求药使求取**〈长生药〉**，获得长生后因出现各种问题开始反对**丰饶星神药师**，最终促成了内部一位将军升格为**巡猎星神岚**。

仙舟联盟与丰饶之民展开了**三次丰饶战争**，使得无数长生世界毁灭，成功威慑了其他文明想要求取长生药的想法。仙舟也损失了几艘仙舟。

## 七、第三次繁荣、毁灭与星核危机

**破坏力**：部分文明世界 | **影响力**：星核阻断了星轨

宇宙总体稳定，一些新派系加入银河社会，进入第三次繁荣。但开拓的无名客长时间沉寂，宇宙旅行彻底被大派系掌控。

距今约3000年前，帝皇战争的遗留及寰宇蝗灾的孑遗破坏了一颗星球，一位失去希望的人觉醒了毁灭命途，成为**毁灭星神纳努克**。祂组建了**反物质军团**开始毁灭宇宙文明。

与此同时，**星核**灾害爆发，进一步加快了文明毁灭，阻断了星际旅行，第三次繁荣结束。

# 第三篇：星神详解

## 现存星神

### 纳努克（毁灭）
诞生于沦亡世界亚德丽芬，以黝黑皮肤、金黄双眼的人类男子形象现身。认为"宇宙的诞生是一种错谬"，统领反物质军团，致力于毁灭一切文明。距今约3000年前由一位失去希望的人觉醒毁灭命途而成神。

### 岚（巡猎）
被仙舟联盟尊称为"帝弓司命"。仙舟文明最初作为求药使求取长生药后，因长生带来种种问题而反对丰饶星神，最终仙舟内部一位将军升格为巡猎星神。无止尽游荡诸界之间，铲除丰饶星神造就的孽物。

### 博识尊（智识）
原为用于求解万物的星体计算机，由机器升格而成。试图体认宇宙、解开万物之谜。祂将宇宙的每一步都计算出来，甚至连帝皇战争和权杖系统的出现都是祂计算干涉引导的结果。曾在某个时间点突然停止了计算，原因不明。

### 希佩（同谐）
来自多个谐乐天体世界的集群星神，千面一体的存在，宣唱着和谐一致的喜乐。吞噬了秩序星神太一，将秩序纳入同谐的范畴。

### 克里珀（存护）
被星际和平公司尊称为"琥珀王"。天慧星墙、亚空晶壁、巨引源基盘的砌造者，建起高墙阻挡贪饕。曾带领众多星神将繁育命途敲碎。

### 药师（丰饶）
众生哺育者，乐土之神，广布不死赐福。被岚视为死敌，仙舟联盟与丰饶之民的战争即因药师的长生赐福引发的各种问题而起。

### 阿哈（欢愉）
宇宙乐子人，鼓励信徒享受欢愉。曾经登上过存在之树。帝皇鲁伯特觉醒时，神秘星神迷思拼命阻止，阿哈却放声大笑，因为"有趣的事情要发生了"。

### 奥博洛斯（贪饕）
执掌贪饕命途的古老星神，永无餍足的世界吞噬者。传说既是星神也是远古生命体"古兽"——黄昏战争中几乎灭绝的古兽中唯一升格为星神的存在。寰宇蝗灾期间乘机吞入被毁灭的世界。

### Ⅸ（虚无）
执掌虚无命途的星神。曾无意义地瞥视一位征服世界的"王"，使其成为自灭者。

### 浮黎（记忆）
执掌记忆命途的星神。公司创始人路易斯·弗莱明退位时，浮黎前来见证。

### 阿基维利（开拓）
从孤绝世界裴伽纳出发，不断开拓未知边界。在星海间铺下银轨，让一个又一个世界彼此相连。因意外而终结命运（已陨落），但开拓的意志仍被星穹列车所继承。

# 第四篇：主要势力

## 1. 星穹列车

继承开拓星神阿基维利意志的旅者组织。乘坐星穹列车在宇宙中旅行，以解决星核危机为己任。列车组就像一个大家庭，成员之间感情深厚。

**核心成员**：开拓者、三月七、丹恒、姬子、瓦尔特、帕姆（列车长）

## 2. 星核猎手

神秘的高手组织，追随**艾利欧**预见的「未来」而行动。艾利欧能够预见未来，他的预见从未出错。星核猎手的每一个行动都是为了到达艾利欧预见的那个「未来」。

**核心成员**：艾利欧（领袖）、卡芙卡、银狼、刃、流萤
**前成员**：开拓者（曾为星核猎手成员）

## 3. 仙舟联盟

由六大仙舟组成的庞大势力，追随巡猎星神岚，以铲除丰饶孽物为使命。仙舟是巨大的星舰型城市，漂航于无尽星海中。

**六大仙舟**：罗浮、曜青、朱明、玉阙、虚陵、方壶
**已损失仙舟**：圆峤（被红巨星吞噬）、苍城（被活化行星噬界罗睺吞没）

## 4. 星际和平公司（IPC）

信奉存护星神克里珀的庞大商业帝国，业务遍布银河。公司是宇宙财富的最终归宿，掌控着绝大多数星际贸易。公司内部有多个部门，各自有不同的职能和利益。

**主要部门**：战略投资部、市场开拓部、防卫部等

## 5. 天才俱乐部

追随智识星神博识尊的精英组织。只有被博识尊"瞥视"——即被博识尊认可的天才才能加入。成员编号代表加入顺序。

**已知成员**：黑塔（#81）、阮·梅（#83）、螺丝咕姆、斯蒂芬、赞达尔·壹·桑原（#1，来古士真身）

## 6. 假面愚者

追随欢愉星神阿哈的组织，追求极致的欢愉和乐趣。成员行事不拘一格，经常以出人意料的方式出现。

**已知成员**：桑博、花火

## 7. 流光忆庭

追随记忆星神浮黎的组织，收集宇宙中的记忆。忆者能够进入忆域，读取和保存各种记忆。

**已知成员**：黑天鹅

## 8. 反物质军团

毁灭星神纳努克统领的军队，致力于毁灭宇宙中的一切文明。是星铁宇宙中最具威胁的敌对势力。

## 9. 丰饶之民

追随丰饶星神药师的势力，信奉长生不老。药师的长生赐福虽然能让人永生，但往往伴随着变异和疯狂，最终变成失去理智的孽物。

## 10. 黄金裔（翁法罗斯）

翁法罗斯的特殊存在，是泰坦火种的继承者。每位黄金裔都承载着一位泰坦的神权，拥有强大的力量。他们肩负着讨伐叛乱泰坦、夺取火种的"逐火之旅"使命。

**十二位黄金裔**：阿格莱雅、缇宝、万敌、遐蝶、那刻夏、风堇、赛飞儿、白厄、昔涟、缇安、缇宁、格奈乌斯

# 第五篇：主要星球与世界

## 1. 空间站「黑塔」

天才俱乐部成员**黑塔**主导建立的博物馆式星际舰船，发射至行星轨道进行研究。

- 拥有数不胜数的奇物"藏品"，是名副其实的星空博物馆
- 大量随舰研究人员定居于此
- 从银河各处收集遗器和奇物进行研究
- 设有谒见系统，可以尝试与博识尊交流

## 2. 雅利洛-Ⅵ（贝洛伯格）

**历史**
- 历史可追溯到上千年前的神话战争——佩伦人十一国与维莱斯集团军的战争
- 后完成从刀耕火种到星际航行的发展，建立了辉煌文明
- 因资源枯竭、星核坠落以及寒潮降临，辉煌文明被葬送
- 寒潮中幸存的难民建造了避难所城市"贝洛伯格"

**社会结构**
- 分为**上层区**和**下层区**，存在严重的阶级分化
- 上层区由大守护者统治，生活相对安定
- 下层区资源匮乏，人民生活艰苦
- 因封闭的自然环境，在星核出现后与其他世界失去联系

## 3. 仙舟「罗浮」

仙舟联盟的六大座舰之一，漂航于无尽星海中，如无回的神矢射向巡猎之主的宿敌。

**历史**
- 与丰饶之民的战争告一段落后，洄航在众多文明昌盛的星域
- 与星际和平公司签署互市协定
- 帮助许多曾被丰饶之民摧毁的世界修复生态
- 从未放松对丰饶之民的戒备

**重要地点**
- **玉界门**：正式抵达仙舟的入口
- **宣夜大道**：星槎海中枢的繁华街道
- **星槎码头**：运送求药、治学和经商的人们
- **司辰宫**：罗浮天舶司总部
- **金人巷**：美食商业街
- **绥园**：狐人巧匠打造的园林洞天
- **鳞渊境**：持明胜境，封印建木玄根
- **幽囚狱**：仙舟监狱系统

**重要组织**
- **云骑军**：仙舟的军事力量，由将军统领
- **太卜司**：负责占卜和预测
- **十王司**：负责审判和执法
- **天舶司**：负责航运和贸易
- **丹鼎司**：负责医药和研究

## 4. 匹诺康尼

被称为**"梦想之地"**的星球，是游戏中最独特和令人不安的地点之一。

**特点**
- 被营销为奢华的度假胜地
- 基于梦境的结构，现实与幻觉模糊
- 十二梦境各有不同的时间设定——如"黄金的时刻"时间永远停留在午夜，"朝露的时刻"最为寂静
- 家族的官邸与办公地点设于"朝露的时刻"
- 探索自由、操纵和逃避现实的代价

**统治家族**
- 橡木家系：星期日为家主

## 5. 翁法罗斯

3.0版本引入的新世界，被称为**"永恒之地"**。这是一个连开拓星神阿基维利也未曾到过的世界。

**核心设定**
- **三重命途缠绕之地**：分别由「记忆」「智识」和一个未知的命途包裹
- **泰坦体系**：翁法罗斯由多位泰坦统治，每位泰坦代表一种理念和力量
- **黄金裔**：泰坦火种的继承者，肩负逐火之旅的使命
- **再创世轮回**：翁法罗斯经历了多次再创世，每次轮回都有不同的命运

**主要城邦**
- **奥赫玛**：圣城，黄金裔的庇护所，繁华祥和
- **悬锋城**：移动堡垒，万敌的故乡，尚武之城
- **雅努萨波利斯**：缇宝的故乡，三相圣地
- **神悟树庭**：那刻夏所在的学术之地
- **斯缇科西亚**：死亡之泰坦的领域，亡魂聚集之地

**主要泰坦**
- **尼卡多利**（纷争）：叛乱泰坦，袭城者，拥有不死神躯
- **欧洛尼斯**（岁月）：古老泰坦，守护记忆碎片
- **瑟希斯**（理性）：曾化名卡吕普索
- **塞纳托斯**（死亡）：已殁于冥河
- **刻法勒**（天空）：在泰坦断崖

## 6. 二相乐园（哈托彼亚）

4.0版本引入的全新世界，位于**失魂星域**的天体。这里是**欢愉星神阿哈的专属领地**，也是星穹列车曾坠毁又再度启航的传奇之地。

**核心设定**
- **"二相"的由来**：整个世界曾被一位绘师施展神迹，收入「画卷」中变成二维平面以躲避毁灭的军锋。即便经改造恢复三维，仍留有二维入口、靠愿力存续的幻造种等二维相关痕迹
- **幻造种与愿力**：自人类的虚构行为中诞生的智慧生灵称为「幻造种」，依靠人们的想象力与关注力——「愿力」而存在。在二相乐园，心愿是一般等价物
- **幻月**：相传阿哈在天空画下一轮有生命的「幻月」，使其成为第一个幻造种。后有八位悲悼伶人向幻月献上惊世骇俗的故事，以求谒见星神
- **幻月游戏**：每逢幻月满盈，以谒神为名的「幻月游戏」如期开始。夺得面具的8位玩家成为「谒者」，积累民众愿力，最终脱颖而出者将得到「欢愉」垂迹，面见阿哈并分享其权柄一分钟

**历史沿革**
- 阿哈创造幻月，建立起包括**古弁才天国**在内的众多文明
- 毁灭的军锋席卷失魂星域，将众多世界化为白地
- 一位绘师施展神迹，将整个世界收入「画卷」拯救众生
- 星穹列车曾坠毁于哈托彼亚海岸，随后与世界一同沉入画中
- 相传阿哈制造的一场爆炸意外促成了列车重生
- 若干年后，废弃的列车得到修复，在姬子的率领下再度启航
- 琥珀纪2155纪，星际和平公司通过「诸世界重建计划」将藏入画中的世界打捞回现实
- 公司利用朋克洛德的以太编辑技术完成重建，派遣高管担任首席执行官
- 二相乐园成为公司重要的资产星球

**主要地区**

*喜笑*
- **二维市**：可能是宇宙中最"欢愉"的城市，包含泊地站、市中心、老饕小巷等
- **绘世学院**：孕育无数绘师人才的顶尖美术学院，得名于曾将世界收入画中的绘师之祖「绘世」
- **鸽川区**：据传为姬子的老家所在，有鸽川大道、共愿大厦等地

*悲泣*
- **幻月秘庭**：接近幻月的塔顶，幻月游戏的中心
- **「世界尽头」酒馆**：饮品均由笑话酿成，包含极乐之星、致命玩笑、黄金告解室、大密室等区域

**重要组织**
- **幸福互助会/幸福微笑研究会**：由「满愿」创办，吸纳20万民众，表面追求幸福，实际利用倏忽细胞干扰认知
- **共愿帮**：活跃于鸽川区的势力
- **破晓战队**：反抗满愿统治的力量

**隐藏秘密**
- 二相乐园封印着**贪饕星神奥博洛斯**的身躯
- 「告死魔」惨案的真相：十五年前的幻月游戏中犯下累累罪行的杀人犯
- 「满愿」继承了来自绝灭大君「归寂」转交的丰饶力量
- 「倏忽」细胞被用于「幸福手术」洗脑民众
- 绯英的本体是**丰饶令使**
